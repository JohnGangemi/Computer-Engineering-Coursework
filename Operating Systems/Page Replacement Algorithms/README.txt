Compilation:
1. Open the terminal and locate the directory named 'project3', excluding the '' characters

2. Type the following command in the terminal to create an executable file: make

3. Run the executable file by typing the following command in the terminal: 
   ./memsim <file.trace> <frames> <rand|fifo|lfu|mfu|lru|opt> <debug|quiet>

Notes:
The 'memsim' program accepts four command line arguments and will terminate appropriately
if the condition is not satisfied.
